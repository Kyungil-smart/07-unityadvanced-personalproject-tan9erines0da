# [기획서] 획득물(Collectible Items) 

## 1. 개요
* **목표**: 맵 곳곳에 배치된 획득물을 통해 플레이어의 이동 경로를 가이드하고, 획득 시 즉각적인 보상과 시각적 피드백을 제공하여 플레이의 리듬감을 형성함.
* **핵심 지표**: 누적 재화(Coin)와 현재 판의 점수(Score) 두 가지 카테고리로 관리함.

## 3. 시각 및 연출 가이드

### 3.1 노란 보석(Giant Yellow Gem) 특수 연출
* **크기(Scale)**: 일반 아이템 대비 약 1.5배 ~ 2.0배 크기로 설정하여 시각적 존재감 강조
* **효과(Effect)**: 획득 시 전용 사운드(웅장한 느낌)와 함께 더 화려한 파티클 연출

### 3.2 공통 연출
* **충돌 판정**: `Trigger`를 사용하여 플레이어 이동 흐름(Momentum)을 방해하지 않음
* **획득 피드백**: 획득 즉시 오브젝트는 비활성화(`Active False`)하며 짧은 획득 SFX 재생

---

## 4. 개발 가이드 (System Architecture)

### 4.1 데이터 관리 (ScriptableObject)
* `ItemDataSO` 클래스를 생성하여 점수 값, 코인 값, 스프라이트, 시각적 스케일(Visual Scale)을 데이터화할 것

### 4.2 상호작용 구조
* `BaseItem` (추상 클래스)를 상속받아 아이템 종류별로 데이터를 주입받아 동작하게 함
* 플레이어의 `IsInvincible` 상태와 무관하게 획득 가능하도록 설정
